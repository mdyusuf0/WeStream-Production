# WeStream Production Blueprint

## WeStream Information Architecture (Provisional Sitemap)

- **Home (`/`)**: High-impact introduction to the studio. Primary CTA: "View Showreel" / "Start a Project". (PROVISIONAL)
- **About (`/studio` or `/about`)**: Team, mission, production philosophy. (BLOCKED BY CLIENT CONTENT)
- **Services Listing (`/services`)**: Overview of capabilities (Editing, VFX, Motion, Color). (BLOCKED BY CLIENT CONTENT)
- **Service Details (`/services/[slug]`)**: Deep dive into specific offerings with SEO FAQs and related work. (BLOCKED BY CLIENT CONTENT)
- **Work / Portfolio (`/work`)**: Gallery of projects. (BLOCKED BY CLIENT CONTENT)
- **Case Study (`/work/[slug]`)**: Detailed breakdown of a specific project (Challenge, Execution, Result). (BLOCKED BY CLIENT CONTENT)
- **Contact (`/contact`)**: Lead generation form and studio info. (CONFIRMED)

## Homepage Blueprint (Section-by-Section)

1. **Hero (Cinematic Reel)**: Full-bleed autoplay background video. Minimal text. Premium typography. CTA to view full showreel or contact.
2. **Featured Work**: Large, immersive, hover-activated project previews. Spatial gallery feel.
3. **Studio Capabilities (Services)**: Concise list of core disciplines with micro-animations.
4. **Social Proof (The WeStream Impact)**: Selected logos and highly curated, high-end testimonials.
5. **Insights / BTS**: Behind-the-scenes looks or blog highlights (optional).
6. **Footer**: Clean, architectural footer with deep links for SEO (Services, Locations) and Contact CTAs.

## Advanced Experience Opportunities

1. **Cinematic Project Transitions (HIGH PRIORITY)**: Fluid, page-to-page transitions (e.g., image expanding to full screen) using View Transitions API or Framer Motion to make browsing the portfolio feel like watching a film.
2. **Video Hover Previews (HIGH PRIORITY)**: Project thumbnails silently play high-quality video snippets on hover.
3. **Aperture/Lens-inspired Transition (EXPERIMENT ONLY)**: A custom WebGL or CSS mask transition mimicking a camera lens for the hero section.
4. **Spatial Project Gallery (HIGH PRIORITY)**: A smooth, non-linear scroll or drag-to-explore grid for the main work index, breaking out of standard boxy CSS grids.
5. **Interactive Showreel (EXPERIMENT ONLY)**: A showreel where users can click on specific shots to branch into the respective case study.
