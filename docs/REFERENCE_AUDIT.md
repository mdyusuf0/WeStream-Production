# Reference Audit

**Primary reference:** https://www.moshimoshi.in/ (Main website system)

**Secondary reference strategy:** Study relevant internal public routes from the primary site to understand the complete design/content system.

**Specific known internal example:** https://www.moshimoshi.in/locations-video-production-company-bangalore (Useful for studying long-form service content, local SEO, FAQs, and portfolio integration. It must not define the entire WeStream website architecture by itself).

## Rules

- Inspiration only
- No cloning
- No scraping
- No copied layout, source code, text, animations, or visual identity

## Full-Site Audit Requirement

Do not audit only the homepage visually. Create a route inventory of relevant publicly accessible pages discovered from the primary site.
For each verified route, record: URL, page type, purpose, major sections, shared components, unique components, content depth, conversion pattern, media usage, motion behavior (if verified), responsive behavior (if verified), SEO role (if inferable).

## Audit Statements

- **Observed:** The reference site uses portfolio-led storytelling and heavy media integration.
- **Inferred:** The target audience values visual proof of capabilities over text-heavy descriptions.
- **Proposed:** Implement a cinematic media-first layout that prioritizes high-quality video/image display without blocking the core content flow.
