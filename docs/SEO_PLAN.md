# SEO Plan

SEO must be architectural, not an afterthought.

## Core Implementation

- Proper metadata using Next.js Metadata API.
- Semantic HTML heading hierarchy (single `<h1>` per page).
- Crawlable, clear content structures.
- Canonical URLs on all pages.

## Future Schemas & Features

- `sitemap.xml` and `robots.txt` generation.
- Open Graph and Twitter sharing metadata.
- Organization Schema (JSON-LD).
- VideoObject Schema for hosted project media (where appropriate).
- Breadcrumb Schema.

> [!CAUTION]
> Do not create spammy location pages or keyword-stuff. Quality over quantity.
